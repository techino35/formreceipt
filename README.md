# FormReceipt

Googleフォームの送信をトリガーに、回答内容をGoogle Docsテンプレートに差し込んでPDF生成・メール送付・Drive保存を自動化するGoogle Apps Scriptアドオン。

## Features
- フォーム送信時に自動でPDF生成
- {{質問タイトル}} プレースホルダーで差し込み
- 回答者へ受領確認メール（PDF添付）
- Drive自動保存 `/FormReceipt/[フォーム名]/[日付]_[連番].pdf`
- Free: 月10件 / Pro: 無制限 + ロゴ挿入
