# FormReceipt - Google Workspace Marketplace

## Short Description
Automate form receipt emails. Generate PDFs from Google Forms responses and email them instantly.

## Full Description
FormReceipt automatically generates PDF receipts from Google Form submissions and emails them to respondents.

### Features
- Template-based PDF generation
- Auto email with PDF attachment
- Drive organized storage
- Sequential receipt numbering
